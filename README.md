# BookStore
Simple and one file book store application:

<b>Supported platforms:</b><br>

- Linux (currently)

<b>Requirements:</b><br>

- Python 3+ <br>
- Gtk 3.0 <br>
- Postgresql 14 <br>

You need to create database first then run app.
You can edit username and database to your liking in file.
<br>

<b>Features: </b>
 - Add new book
 - Remove
 - Edit (double click to edit)
